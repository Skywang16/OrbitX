## 关于 Eko

Eko 是一个强大的框架，专为构建生产就绪的代理工作流而设计。它提供了高效的跨平台解决方案，用于自动化工作流的规划和执行。

### 主要特性

- 🎯 **多代理协作** - 支持多个代理协同工作
- 🌐 **跨平台支持** - 浏览器扩展、Node.js、Web 环境
- 🔧 **高度可定制** - 灵活的代理和工具系统
- 📊 **生产就绪** - 企业级可靠性和性能
- 🔄 **流式回调** - 实时监控和人工干预
